# Expense Tracker

A mobile expense tracking app built with React Native and Expo. Track your spending, and stay on top of your budget — all synced in real time via Firebase.

---

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | React Native 0.85 + Expo 56 |
| Language | TypeScript |
| Navigation | React Navigation (Stack + Bottom Tabs) |
| State Management | React Context API |
| Backend / Database | Firebase (Firestore + Auth) |
| HTTP Client | Axios |
| Icons | Lucide React Native |

---

## Prerequisites

- Node.js 18+
- Expo CLI (`npm install -g expo-cli`)
- Expo Go app on your device, or a running iOS/Android simulator
- A Firebase project with Firestore and Authentication enabled

---




## Features

- Add, edit, and delete expenses
- Real-time sync with Firebase Firestore
- Clean bottom-tab navigation
